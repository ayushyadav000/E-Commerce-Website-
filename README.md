# 🛒 Modern E-Commerce Website

Modern and responsive E-Commerce website built using React, TypeScript, Vite, and Tailwind CSS.  
It provides a premium shopping experience with smooth animations, responsive design, and modern UI components.

---

# 🚀 Features

✅ Modern UI/UX Design  
✅ Fully Responsive Layout  
✅ Fast Performance using Vite  
✅ React Router Navigation  
✅ Redux Toolkit State Management  
✅ Smooth Animations using Framer Motion  
✅ Mobile Friendly Design  
✅ Reusable Components  
✅ Clean Folder Structure  
✅ Single File Build Support  

---

# 🛠️ Technologies Used

## Frontend
- React 19
- TypeScript
- Vite
- Tailwind CSS

## State Management
- Redux Toolkit
- React Redux

## Routing
- React Router DOM

## Animation
- Framer Motion

## Icons
- React Icons

## Utility Libraries
- clsx
- tailwind-merge
